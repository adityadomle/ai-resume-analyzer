# Echokart TechStore

A responsive eCommerce platform for tech products, built with React.js and Tailwind CSS.

---

## Features

- Clean and responsive UI  
- Product listings with categories  
- Page routing using React Router  
- Modular component structure

---

## Tech Stack

- React.js  
- Tailwind CSS  
- React Router DOM

---
